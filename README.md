# OSS2025
## Submission to the MICCAI OSS2025 challenge (inference only)
### Preparation
- Download files from the following URL and place these in ./videomae_base
  - https://muroran-it-ac.box.com/s/hd64sdi4fy6l1ogzsvoqrcte3523whet
- Download files from the following URL and place these in ./resources
  - https://muroran-it-ac.box.com/s/iww7znwz1p9v57ngd81q8w62nmceu36f
- Create directories 'input' and 'output'
- Place mp4 files into 'input' directory
### Run
- $ python inference.py GRS input output
- $ python inference.py OSATS input output
